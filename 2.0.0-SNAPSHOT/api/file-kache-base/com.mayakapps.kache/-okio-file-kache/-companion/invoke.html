Opens the persisted Least Recently Used (LRU) cache in the provided directory or creates a new one if it doesn't already exist.

Parameters

directoryPath

The directory for storing the journal and cached files. To prevent data loss, it shouldn't be used for storing files not managed by this class.

maxSize

The max size of this cache in bytes, excluding the size of the journal file.

creationDispatcher

The coroutine dispatcher used for executing creationFunction of put requests.

function used for transforming keys to be safe for filenames. See KeyTransformer
